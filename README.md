# Operations & Supply Chain Analytics

Python-based operations analytics project covering inventory optimization, queueing analysis and project scheduling for a supply chain case study.

This project was developed as part of graduate-level coursework in Operations Management and Logistics. The analysis focuses on supporting operational decision-making for a start-up case company by applying quantitative methods from Operations Research.

## Project Scope

The project consists of three main parts:

### Inventory Optimization
- Evaluated inventory policies for multiple SKUs
- Applied EOQ, (Q,R) and Newsvendor models
- Assessed demand uncertainty, ordering costs, holding costs, shortage costs and safety stock requirements

### Queueing Analysis
- Analyzed the picking–packing service system during peak demand periods
- Evaluated system stability under different staffing levels
- Assessed waiting line behavior and service level performance
- Identified a staffing policy that improves operational reliability

### Project Scheduling
- Applied CPM to analyze project duration and critical path activities
- Evaluated crashing decisions to reduce project duration
- Supported go-live planning while controlling additional cost
