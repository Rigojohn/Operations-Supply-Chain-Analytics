README – Οδηγίες Εκτέλεσης Project

Για την εκτέλεση των Jupyter Notebooks του project απαιτείται Python 3
και οι παρακάτω βιβλιοθήκες:

- pandas
- scipy
- openpyxl

Εγκατάσταση βιβλιοθηκών (μέσω pip):
pip install pandas scipy openpyxl

Εναλλακτικά, σε περιβάλλον Anaconda:
conda install pandas scipy openpyxl

Οι υπόλοιπες βιβλιοθήκες που χρησιμοποιούνται είναι ενσωματωμένες
στην Python και δεν απαιτούν επιπλέον εγκατάσταση.

Οδηγία για το notebook "ΟυρέςΑναμονής.ipynb":
Στο notebook αυτό, ο χρήστης μπορεί να δοκιμάσει διαφορετικό αριθμό
εξυπηρετητών (c), εισάγοντας την τιμή όταν ζητηθεί από το πρόγραμμα.
Για την εκτέλεση κάθε παραδείγματος αρκεί να πατηθεί Shift + Enter
στο αντίστοιχο κελί.
